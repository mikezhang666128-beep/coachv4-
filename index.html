What is N / W / D / F?
Tap the chip on any set to cycle through these types.
N
Normal
Your regular working set. Counts toward total volume and triggers the rest timer.
W
Warmup
Light sets to prepare. Doesn't count toward volume. Skips the rest timer.
D
Drop set
After a working set, drop weight and rep again with minimal rest. Counts as extra intensity.
F
To failure
You went until form broke. Use sparingly — the coach uses this to detect overreaching.
